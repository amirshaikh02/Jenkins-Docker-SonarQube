<H1>Automated CI/CD Pipeline for Portfolio Website using Jenkins and Docker on AWS EC2</H1>

<h3>
  Tech Stack used: AWS EC2, Jenkins, Docker, SonarQube, Ubuntu, GitHub, HTML, CSS, JS.
</h3>

<h6>Designed and implemented a CI/CD pipeline using Jenkins on AWS EC2 with Ubuntu OS. </h6>
<h6>Configured three EC2 instances to host Jenkins, SonarQube, and Docker services.</h6>
<h6>Integrated GitHub repository to automatically pull portfolio website code on every update.</h6>
<h6>Performed static code quality and security analysis using SonarQube during the build process.</h6>
<h6>Containerized and deployed the application using Docker for reliable and faster deliveryTech</h6>
